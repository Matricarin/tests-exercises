**Задача: Middleware для метрик времени**

* **Задача:** 

Напиши кастомное `Middleware`, которое замеряет время выполнения каждого HTTP-запроса и пишет в лог (через `ILogger`): 
`[Method] [Path] took [Time] ms`.

* **Валидация:** 

Сделай GET-запрос через Postman/curl. В консоли приложения должна появиться строчка с точным временем миллисекунд.
