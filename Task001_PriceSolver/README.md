📋 Условие задачи

Дан массив чисел a длины n. 
Необходимо вернуть две пары индексов элементов массива, так чтобы:

    Первая пара (i1​,j1​), где i1​<j1​, для которой разность ai1​​−aj1​​ минимальна.
    Вторая пара (i2​,j2​), где i2​<j2​, для которой разность ai2​​−aj2​​ максимальна.

⚠️ Важное ограничение 

В обеих парах первый индекс должен быть строго меньше второго (i<j). 
Это означает, что ai​ — это элемент, который стоит в массиве левее, а aj​ — элемент, который стоит правее. 
Мы всегда вычитаем из «левого» элемента «правый».

Если в массиве есть несколько разных пар индексов, которые дают одинаковую минимальную (или максимальную) разность, нужно выбрать ту пару, у которой:

    Меньше индекс i (левый элемент стоит как можно раньше).
    Если левые индексы i одинаковые, то выбираем ту пару, у которой меньше индекс j (правый элемент стоит как можно раньше).

🔍 Пример 

Возьмем массив: [4, 1, 4] 

Ищем глобальный минимум разности:
Минимальное значение среди разностей (3,0,−3) — это −3. Оно достигается на паре индексов 2 и 3.

Ищем глобальный максимум разности:
Максимальное значение среди разностей (3,0,−3) — это 3. Оно достигается на паре индексов 1 и 2.

    Ответ: 2 3 1 2

**Что важно:**
Правило Equality, Range, Covering.
