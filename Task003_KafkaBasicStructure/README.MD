**Контекст:** Напиши минимальный рабочий каркас для отправки и получения сообщений, используя официальную библиотеку `Confluent.Kafka`. Это покроет 90% практических вопросов на интервью для мидла.
**Задача:** Напиши два изолированных метода:

1. **Продюсер:** Метод `Task SendOrderMessageAsync(string orderId, string status)`, который конфигурирует продюсера, сериализует данные в JSON и отправляет сообщение в топик `"orders-topic"`.
2. **Консьюмер:** Метод `void StartConsuming()`, который запускает бесконечный цикл чтения из топика `"orders-topic"`.

* *Важное условие:* Отключи автокоммит офсетов (`EnableAutoCommit = false`) и реализуй **ручной коммит** строго после того, как сообщение успешно вывелось в консоль (семантика *At-least-once*).
